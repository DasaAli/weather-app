## 💕 Features

- Search weather by city 🌆  
- See temperature 🌡️, description 📝, and weather icon 🌤️  
- Soft pastel pink pixel art theme 🍬  
- Responsive design for desktop & mobile 📱  
- Powered by [OpenWeather API](https://openweathermap.org/api) ☁️  

---
